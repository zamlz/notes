Elisp is a dialect of the [[file:lisp_programming_language.org][Lisp Programming Language]] that is used to create the integrated environment of [[file:../config/emacs.org][GNU/Emacs]]. It is a fully fledged programming language and is desgined to make the editor extensible to any degree.
