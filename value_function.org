The value function is a predicition of total future [[id:ed78ebfa-633f-44f3-bb4f-d6c946df8386][Reward Signal]]s. This function is used evaluate how good or bad a state in [[id:9e3eca4d-4109-46f9-8632-45c02dcead4e][Environment]] is. It be defined as,

\[v_{\pi}(s) = \mathbb{E}_{\pi}[R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \dots | S_t = s]\]
