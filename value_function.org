The value function is a predicition of total future [[file:reward_signal.org][Reward Signal]]s. This function is used evaluate how good or bad a state in [[file:rl_environment.org][Environment]] is. It be defined as,

\[v_{\pi}(s) = \mathbb{E}_{\pi}[R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \dots | S_t = s]\]
