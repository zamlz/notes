An [[file:rl_environment.org][Environment]] where the [[file:rl_agent.org][Agent]] is not able to see the true state information is a partially observable environment (POMDP).

In these scenarios, it cannot be guaranteed that the agent state is a [[file:markov_state.org][Markov State]].
